### 範例執行

C# 範例程式碼以 PHP 改寫

- PHP 版本 >= 7.1
- PHPUnit 版本 7.4.3

安裝必要套件

```bash
composer install
```

執行測試

```bash
./vendor/bin/phpunit
```
